  • 3학년 1학기 (현재)

    1. 알고리즘
      2. 2학년 때 배웠으나, 군대에 다녀오느라 다 까먹어버렸습니다.
      그래서 기초부터 다시 공부하고 있습니다.
      inflearn 강의와 하루에 1~2문제 예제 문제 풀이를 통해 기초부터 다시 공부하는 중입니다.

    2. C++ 기초 문법
      3. C++은 언리얼 엔진 스크립트 작성 언어이며, 알고리즘 문제 풀이에 필요합니다.
      2학년 때 깊게 배우지 않았던 STL 문법의 기초와, 까먹은 내용을 다시 공부하고 있습니다.
      inflearn 강의를 활용하고 있으며, 알고리즘 공부에 계속 C++언어를 사용하며 익숙해지고 있습니다.


  • 3학년 여름방학

    1. 알고리즘
      2. 알고리즘 실력은 문제를 많이 풀어 보는게 방법이라고 들었습니다.
      1학기 기간에 알고리즘 기초 공부를 마치고, 방학 기간에 ‘백준’, ‘프로그래머스’ 등의 알고리즘 풀이 사이트를 통해 계속 알고리즘 문제를 접해보고 실력을 늘려나갈 생각입니다.

    2. 언리얼 엔진 기초
      3. 학기 중에는 수업, 과제 등으로 공부량이 많은 공부는 진행하기 어려웠습니다.
      방학 기간을 통해 공부량이 꽤 되는 언리얼 엔진 기초 공부를 진행할 생각입니다.
      class101 ‘러셀‘의 언리얼 엔진 강의, 언리얼 엔진 공식 홈페이지의 ’학습‘등을 이용할 계획입니다.


  • 3학년 2학기

    1. 알고리즘
      2. 이쯤부터는 behavior tree, 다익스트라 알고리즘, A*알고리즘 등 게임에서 자주 사용되는 알고리즘을 공부할 생각입니다.

    2. 언리얼 엔진
      3. 여름방학 기간에 언리얼 엔진 기초 공부가 끝난다고 생각하지 않습니다.
      개강 후에도 꾸준히 공부할 계획이며, 기초적인 공부가 완료되었다고 생각하면 class101 강의뿐만 아니라 inflearn 강의, 책 등을 활용하여 공부할 계획입니다.


  • 3학년 겨울방학

    1. 알고리즘
      2. 알고리즘에는 끝이 없습니다.
      실제 게임회사에서 진행한 코딩 테스트 문제와 그와 유사한 문제들, 이전에 풀면서 어려웠던 문제들을 풀이해볼 생각입니다.

    2. 언리얼 엔진
      3. 이쯤엔 언리얼 엔진을 통해 저만의 게임을 하나 만들어 볼 생각입니다.
      취업 시 준비해야 할 포트폴리오 수준까지는 아니더라도, 언리얼 엔진이 손에 익도록 계속 공부하며 만들어볼 생각입니다.

    3. C++ 심화 공부
      4. C++ 언어는 계속 발전하고 있습니다.
      실제 업계에서 사용할 수 있는 스킬들을 익히기 위해 ‘Effective C++’같은 C++ 심화 책을 이용하여 공부할 계획입니다.
